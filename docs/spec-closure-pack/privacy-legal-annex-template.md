# Privacy and Legal Annex Template

This artifact must be approved before pilot go-live.

Required sections:
- approved consent copy by event type
- sponsor communication wording
- DSR SLA
- downstream deletion propagation policy for CRM and webhook recipients
- field-by-field anonymization matrix
- retention trigger definition:
  - event end
  - report freeze
- organizer-approved event policy defaults

Operational rule:
- if this annex is incomplete, the event cannot move to go-live approval.
